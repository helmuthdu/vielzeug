# defer

`defer` is not exported by `@vielzeug/arsenal`.

For deferred resolution patterns, use `waitFor` to poll a condition, or `abortable` to attach cancellation to any existing promise.

- [waitFor](./waitFor.md)
- [abortable](./abortable.md)
