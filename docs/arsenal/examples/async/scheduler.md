# Scheduler / polyfillScheduler

`Scheduler` and `polyfillScheduler` are not exported by `@vielzeug/arsenal`.

Use `sleep` with an `AbortSignal` or `waitFor` to schedule work around async conditions.

- [sleep](./sleep.md)
- [waitFor](./waitFor.md)
