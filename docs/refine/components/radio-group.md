# Radio Group

`ore-radio-group` documentation is on the [Radio](./radio.md) page, which covers both `ore-radio` and `ore-radio-group` together.

## What To Import

```ts
import '@vielzeug/refine/radio-group';
```

## Quick Example

```html
<ore-radio-group name="plan" label="Choose a plan" value="free" required>
  <ore-radio value="free">Free</ore-radio>
  <ore-radio value="pro">Pro</ore-radio>
  <ore-radio value="enterprise">Enterprise</ore-radio>
</ore-radio-group>
```

See [Radio (+ Radio Group)](./radio.md) for the full API reference, attributes, events, and examples.
