## List all packages

Tool call:

```json
{ "name": "list-packages", "arguments": {} }
```

Result excerpt:

```json
[
  {
    "name": "@vielzeug/stateit",
    "slug": "stateit",
    "version": "3.0.0",
    "availableDocPages": ["index", "api", "usage", "examples"],
    "hasSource": true
  },
  {
    "name": "@vielzeug/fetchit",
    "slug": "fetchit",
    "version": "3.0.0",
    "availableDocPages": ["index", "api", "usage", "examples"],
    "hasSource": true
  }
]
```
