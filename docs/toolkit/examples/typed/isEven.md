# isEven

Checks if a number is even.

## API

```ts
isEven(value: number): boolean
```

- `value`: Number to check.
- Returns: `true` if value is even, else `false`.

## Example

```ts
import { isEven } from '@vielzeug/toolkit';

isEven(2); // true
isEven(3); // false
```

## Notes

- Only works with numbers.
