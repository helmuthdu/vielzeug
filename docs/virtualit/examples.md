# Virtualit Examples

## How to Use These Examples

These examples cover fixed-height lists, variable heights, grouping, keyboard nav, and infinite scroll.

1. Start with the first example page to learn the baseline pattern.
2. Move to advanced pages that match your framework or production scenario.
3. Keep the API page open while adapting snippets to your project.

These examples are aligned with the current `@vielzeug/virtualit` and `@vielzeug/virtualit/dom` APIs.
