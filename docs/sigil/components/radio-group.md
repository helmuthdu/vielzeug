# Radio Group

`sg-radio-group` documentation is on the [Radio](./radio.md) page, which covers both `sg-radio` and `sg-radio-group` together.

## What To Import

```ts
import '@vielzeug/sigil/radio-group';
```

## Quick Example

```html
<sg-radio-group name="plan" label="Choose a plan" value="free" required>
  <sg-radio value="free">Free</sg-radio>
  <sg-radio value="pro">Pro</sg-radio>
  <sg-radio value="enterprise">Enterprise</sg-radio>
</sg-radio-group>
```

See [Radio (+ Radio Group)](./radio.md) for the full API reference, attributes, events, and examples.
