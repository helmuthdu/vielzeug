# Showcases

End-to-end examples that combine multiple `@vielzeug/*` packages in one real app flow.

## Kanban Demo

A production-style Kanban board demo that combines state, forms, drag-and-drop, routing, persistence, charts, i18n, keyboard shortcuts, undo/redo, and UI components across the Vielzeug stack.

- **Live demo:** [StackBlitz](https://stackblitz.com/edit/vitejs-vite-rhttrmbq)
- **Source code:** [Github](https://github.com/helmuthdu/vielzeug/tree/main/demos/kanban)

### Packages used

`@vielzeug/ore`, `@vielzeug/refine`, `@vielzeug/ripple`, `@vielzeug/dnd`, `@vielzeug/wayfinder`, `@vielzeug/forge`, `@vielzeug/spell`, `@vielzeug/vault`, `@vielzeug/ledger`, `@vielzeug/keymap`, `@vielzeug/prism`, `@vielzeug/lingua`, `@vielzeug/scout`, `@vielzeug/sourcerer`, `@vielzeug/herald`, `@vielzeug/courier`, `@vielzeug/clockwork`, `@vielzeug/tempo`, `@vielzeug/coins`, `@vielzeug/pulse`, `@vielzeug/rune`, `@vielzeug/familiar`, `@vielzeug/flux`.
